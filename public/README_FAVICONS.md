# Favicon Setup

The SEO component expects the following favicon files in the `/public` directory:

- `favicon-16x16.png` - 16x16 favicon
- `favicon-32x32.png` - 32x32 favicon
- `apple-touch-icon.png` - 180x180 Apple touch icon
- `site.webmanifest` - Web app manifest (already created)

## How to Generate Favicons

1. Create a square logo image (at least 512x512px)
2. Use a favicon generator like:
   - https://realfavicongenerator.net/
   - https://favicon.io/
3. Download the generated files
4. Place them in the `/public` directory

## Quick Start

For now, you can use a placeholder by creating a simple colored square PNG with your brand color (#d62828).
